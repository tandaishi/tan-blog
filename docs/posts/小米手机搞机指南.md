# 小米手机搞机指南

## 0 注意事项

搞基有风险，请自负风险，如果觉得自己基础不足，没有信心，可以去看一些linux有关的教程，熟悉熟悉命令行，最好有科学上网的环境，如果你无法科学上网的话，我也会尽可能将相关的安装包上传到网盘。但是我讲解的时候是按照有科学上网的环境讲解的，强烈建议不要在晚上开始搞基。
