# zoomit 使用教程

## 官网下载地址看一下


## ctrl + 1 放大并写画

## ctrl + 2 不放大并写画


## 写画模式下
0 直接滚滚轮，调整视图远近，移动鼠标会调整视口范围，
1 左键单击，显示画笔，右键单击不显示画笔，再右键退出写画模式(esc键也行)，注意只有不显示画笔的情况下视口才会跟随鼠标
2 按住ctrl滚滚轮，放大缩小画笔
3 按t，输入文字，滚轮调整字号
4 按住shift再画，会出直线，按住ctrl再画，会出方框，按住shift + ctrl 再画，会出带箭头的直线，按住 Tab 会出 
5 切换颜色 purple red green blue yellow orange 以上颜色可Shift + 颜色
6 按 w 是白板，按 e 是清除，ctrl + z 可以撤销上一步操作
带着看一下相关设置

## ctrl + 4 鼠标周围放大
期间可用ctrl + 上，或ctrl+ 下，放大缩小
注意该功能想退必须再次按 ctrl + 4 退
注意该模式无法被obs捕捉到，zoomit自己的录屏可以捕捉到

## ctrl + 6 截图
写画模式下可截图
ctrl + c 复制屏幕
ctrl + s 保存屏幕
ctrl + shift + s 就是截图保存

## ctrl + 3 倒记时
滚轮可以增加或减少时间
颜色键可以控制颜色
带看一下设置


## ctrl + 5 录屏
可以 ctrl + shift + 5 录局部
看设置即可
